# Screenshots

Add screenshots of CodexBar Windows here:

1. **tray-icon.png** - The system tray icon showing usage colors
2. **main-panel.png** - The main popup panel with provider usage
3. **settings.png** - The settings dialog
4. **about.png** - The about dialog
5. **cookies.png** - The manual cookie input dialog

## How to Take Screenshots

1. Run `codexbar menubar` to start the GUI
2. Use Windows Snipping Tool (Win+Shift+S) to capture:
   - The tray icon in the system tray area
   - The main popup panel (click the tray icon)
   - Each dialog (Settings, About, Cookies)
3. Save as PNG files with the names above
4. Optionally crop/resize to ~800px width for the main panels
